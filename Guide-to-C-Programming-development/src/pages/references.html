References - Πηγές

Στην σύνθεση του υλικού της σελίδας χρησιμοποιήσαμε, άμεσα ή έμμεσα, την προσωπική μας εμπειρία με την γλώσσα και πληροφορίες απο τις παρακάτω σελίδες, τις οποίες προσαρμόσαμε έτσι ώστε να αποτελούν μία αποτελεσματική και κατανοητή επεξήγησή της. Οι πηγές έχουν ως εξής:

  • The C Programming Language - Brian W. Kernighan and Dennis M. Ritchie
  • The GNU C Reference Manual
  • Why learning c programming is a must?

Παραθέτουμε επίσης την παρακάτω μαγνητοσκόπηση η οποία παρουσιάζει πλήρως την γλώσσα, καθώς και το κρίνουμε χρήσιμο για όσους ενδεχομένως να μην προτιμούν τον γραπτό λόγο ως μέσω διδασκαλίας.
