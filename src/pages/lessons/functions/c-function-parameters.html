Παράμετροι συναρτήσεων

Όπως είδαμε στο προηγούμενο μάθημα, μπορούμε να περνάμε παραμέτρους στις συναρτήσεις ώστε να λειτουργήσει πάνω στα δεδομένα που τους δίνουμε. Ας δούμε τώρα μερικές πιο ειδικές περιπτώσεις παραμέτρων.

Pass by value

Οι γλώσσες γενικά λειτουργούν με δύο μηχανισμούς: pass by value και pass by reference. Το πρώτο σημαίνει πως τα ορίσματα που δίνουμε σε μία συνάρτηση δεν είναι η ίδια μεταβλητή που της περάσαμε, αλλά ένα αντίγραφό της με την ίδια τιμή. Αυτό σημαίνει οτι αν μεταβάλλουμε την τιμή ενός ορίσματος μέσα σε μία συνάρτηση, η τιμή της κανονικής μεταβλητής δεν επρόκειτο να αλλάξει 


  #include <stdio.h>

  void test(int x)
  {
    x = 5;
  }

  int main(void)
  {
    int x = 3;
    printf("%d", x);
    test(x);
    printf("%d", x);
  }
Η έξοδος αυτού του προγράμματος θα είναι 33 όχι 35, επειδή όπως είπαμε η μεταβλητή x που βρίσκεται στο σώμα της test ΔΕΝ είναι η ίδια με αυτή στο σώμα της main. Από αυτή την παρατήρηση απορρέει ένα εύλογο ερώτημα: πώς μπορούμε να μεταβάλλουμε μία μεταβλητή στο σώμα μίας άλλης συνάρτησης; Στο σημείο αυτό πρέπει να θυμηθείτε το προ-προηγούμενο μάθημα με τους δείκτες. Αν περάσουμε έναν δείκτη προς μία μεταβλητή σε μία συνάρτηση, τότε η συνάρτηση θα ξέρει την διεύθυνση στην μνήμη της μεταβλητής που θέλουμε να αλλάξουμε και συνεπώς μπορεί να την αλλάξει χωρίς κανένα πρόβλημα. 

  #include <stdio.h>

  void test(int* x) 
  {
    *x = 5;
  }

  int main(void)
  {
    int x = 3;
    printf("%d", x);
    test(&x);
    printf("%d", x);
    return 0;
  }
Όπως μπορεί να καταλάβατε, το πρόγραμμα αυτό θα εμφανίσει 35
