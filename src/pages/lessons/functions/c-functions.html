Συναρτήσεις

Οι συναρτήσεις είναι κομμάτια κώδικα τα οποία τρέχουν μόνο όταν καλούνται από κάποιο άλλο μέρος του προγράμματος. Οι συναρτήσεις μπορούν να λάβουν δεδομένα ως παραμέτρους, τα οποία μπορούν να χρησιμοποιήσουν για να επιτελέσουν κάποιον σκοπό.

Δημιουργιά συναρτήσεων

Ο τρόπος με τον οποίο συντάσσουμε συναρτήσεις παραμένει γενικά ίδιος. Πρέπει να πούμε στον μεταγλωττιστή τι είδος μεταβλητής επιστρέφει, το όνομα της συνάρτησης, τον τύπο και όνομα κάθε παραμέτρου και το 'σώμα' της συνάρτησης. 

  return_type name(type1 arg1, ...) 
  {
    /* σώμα */
  }
Μπορούμε να το δούμε εύκολα στην πράξη αυτό με την συνάρτηση main την οποία ήδη γνωρίζουμε 
  int main(int argc, char* argv[])
  {
    /* ... */
  }

Κλήση συναρτήσεων

Με τον ίδιο τρόπο που καλούμε τις συναρτήσεις των βιβλιοθηκών (π.χ. printf), μπορούμε να καλέσουμε και τις δικές μας συναρτήσεις. Αρκεί να γράψουμε το όνομα και να περάσουμε τα κατάλληλα ορίσματα. 

  #include <stdio.h>

  /* void σημαίνει 
  δεν επιστρέφει τίποτα */
  void func(int x)
  {
    for (int i = 0; i < 100; ++i)
    {
      printf("%d\n", i);
    }
  }

  int main(void)
  {
    func(5);
    func(10);
    func(); /* error */
  }
