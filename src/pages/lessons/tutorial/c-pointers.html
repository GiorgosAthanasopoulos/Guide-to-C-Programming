Δείκτες

Οι δείκτες είναι ένα ίσως και το πιο σημαντικό μέρος της C και θεωρείται και ως ένα απο τα πιό δύσκολα στην πλήρη κατανόηση. Ας αναλύσουμε τους δείκτες βήμα βήμα

Μεταβλητές στην μνήμη

Η κύρια μνήμη του υπολογιστή αποτελείται απο κελιά τα οποία αποθηκεύουν σχήματα απο bit. Το κάθε ένα απο αυτά τα κελιά έχουν μία διεύθυνση, δηλαδή έναν αριθμό ο οποίος είναι ξεχωριστός μόνο για αυτούς. Με άλλα λόγια, εαν γνωρίζουμε την διεύθυνση ενός κελιού μπορούμε να το βρούμε και να δούμε τι σχήμα bit έχει αποθηκεύσει. Όταν δηλώνουμε μία μεταβλητή στην C η μεταβλητή αυτή χρησιμοποιεί κάποια κελιά μνήμης (το πλήθος εξαρτάται απο τον τύπο της μεταβλητής) στα οποία αποθηκεύεται η τιμή της. Οι μεταβλητές οι οποίες χρησιμοποιούνται για να αποθηκεύσουν μία διεύθυνση κελιόυ λέγονται δείκτες. Μπορούμε να δηλώσουμε έναν δείκτη ως εξής: 

type* i;
Οπού το i είναι ένας δείκτης που κρατάει την διεύθυνση μίας μεταβλητής τύπου type. Για να θέσουμε στην i μιά τιμή χρειαζόμαστε την διεύθυνση μιάς μεταβλητής τύπου type. Για να βρούμε την διεύθυνση μιάς μεταβλητής χρησιμοποιούμε το σύμβολο &: 
int x = 5;
int* p = &x;
Οπότε τώρα το p κρατάει την διεύθυνση του x.

Dereferencing

Ωραία, έχουμε την διεύθυνση της x... Τώρα τι; Υπάρχει ένας τρόπος όταν έχουμε έναν δείκτη που κρατάει την διεύθυνση μιάς μεταβλητής, να πάμε στην διεύθυνση που δείχνει και είτε να διαβάσουμε το περιεχόμενό τής είτε να το αλλάξουμε. Αυτά το κάνουμε με την χρήση του αστερίσκου πρίν το όνομα του δείκτη. 

  int x = 5;
  int* p = &x;
  *p = 3;
  printf("%d", x);
Ενώ φαινομενικά δεν έχουμε αγγίξει την μεταβλητή x, αν τρέξετε τον παρακάτω κώδικα θα δείτε πως εκτυπώνεται η τιμή 3. Αυτό είναι επειδή χρησιμοποιήσαμε την εντολή *p = 3; που σημαίνει "εκχώρησε την τιμή 3 στο κελί του οποίου η διεύθυνση βρίσκεται μέσα στην μεταβλητή p"
