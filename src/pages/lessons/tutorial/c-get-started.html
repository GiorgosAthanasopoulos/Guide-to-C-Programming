Ξεκινώντας

Για να ξεκινήσετε να συνθέτετε κώδικα και να εκτελείτε τα προγράμματα θα χρειαστείτε δύο πράγματα:

  • Έναν επεξεργαστή κειμένου ή ένα διαδραστικό προγραμματιστικό περιβάλλον
  • Έναν μεταγλωττιστή, σε περίπτωση που διαλέξετε έναν απλό επεξεραστή κειμένου
Εναλλακτικά, εφόσον δεν επιθυμέιτε να εγκαταστήσετε κάποιο πρόγραμμα στον υπολογιστή σας, μπορείτε απλώς να αναπτύξετε προγράμματα μέσω του διαδικτύου. Υπάρχουν πολλά διαθέσιμα μέρη για να το κάνετε αυτό, όπως εδώ. Να έχετε υπόψιν σας όμως οτι μέσω διαδικτύου δεν θα έχετε ούτε το εκτελέσιμο πρόγραμμα, θα είναι πολύ πιο αργή η εκτέλεση και δεν θα έχετε πρόσβαση στην επίδραση με τον υπολογιστή σας (π.χ. με αρχεία)

Εγκατάσταση IDE

Μία δημοφιλής επιλογή για IDE είναι το πρόγραμμα γνωστό ώς Code::Blocks, καθώς και είναι αρκετά πιό ελαφρύ από άλλες δημοφιλείς επιλογές (π.χ. Visual Studio). Μπορείτε να βρείτε την πιο πρόσφατη έκδοση του Codeblocks εδώ. Κατεβάστε το αρχέιο mingw-setup.exe το οποίο θα εγκαταστήσει το πρόγραμμα στον υπολογιστή σας

Αφού ανοίξετε το πρόγραμμα, μπορείτε να δημιουργήσετε ένα αρχείο μέσω του μενού στην κορυφή του παραθύρου, File > New > Empty File, και να το αποθηκεύσετε με το κατάλληλο file extension, που στην περίπτωση μας είναι .c

Έπειτα, όταν θα συνθέτουμε τα προγράμματά μας στα αρχεία που θα φτιάχνουμε και θα επεξεργαζόμαστε μέσα στο Codeblocks, θα μπορούμε να πατάμε Build > Build and Run και να τα εκτελούμε!
