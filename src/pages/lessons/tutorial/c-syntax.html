Σύνταξη

Παρακάτω παρουσιάζουμε ένα πολύ βασικό πρόγραμμα, την σύνταξη του οποίου θα αναλύσουμε 


          #include <stdio.h>

          int main()
          {
            printf("Hello, world!");
            return 0;
          }

Επεξήγηση

Γραμμή 1: Χωρίς να εμβαθύνουμε πάρα πολύ με λεπτομέρειες, χρησιμοποιώντας στην κορυφή του αρχείου τον προεπεξεργαστή include μπορούμε να χρησιμοποιήσουμε βιβλιοθήκες που είτε τις παρέχει η γλώσσα είτε κάποιο τρίτο πρόσωπο. Στην περίπτωσή μας, χρησιμοποιούμε την βιβλιοθήκη stdio.h η οποία μας παρέχει τρόπους να εμφανίζουμε μηνύματα στην οθόνη και να διαβάζουμε πληροφορίες που πληκτρολογεί ο χρήστης κτλ.

Γραμμή 2: Μία κενή γραμμή. Η C αγνοεί τις κενές γραμμές, που σημαίνει ότι μπορούμε να αφήνουμε κενά ανάμεσα στον κώδικά μας ώστε να είναι πιο εύκολος ως πρός την ανάγνωση

Γραμμή 3: Σε κάθε πρόγραμμα υπάρχουν συναρτήσεις οι οποίες περιέχουν κώδικα ανάμεσα σε αγκύλες {}. Η συνάρτηση main είναι η συνάρτηση στην οποία ξεκινάει η εκτέλεση του προγράμματος

Γραμμή 4: Η κλήση της συνάρτησης printf (print format) με παράμετρο μία συμβολοσειρά η οποία θα εμφανιστεί στην οθόνη

Γραμμή 5: Η συνάρτηση επιστρέφει την τιμή 0, που για την συνάρτηση main σημαίνει οτι όλα πήγαν καλά κατά την εκτέλεση του προγράμματος
