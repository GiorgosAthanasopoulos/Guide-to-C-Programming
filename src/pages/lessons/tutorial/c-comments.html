Σχόλια

Τα σχόλια μπορούν να χρησιμοποιηθούν ώς μια επεξήγηση του κώδικα στον αναγνώστη, ώστε να τον κάνει πιο ευνόητο. Μπορεί επίσης να χρησιμοποιηθεί ώστε να αποτρέψει την εκτέλεση κάποιου κώδικα προσωρινά.


Στην πράξη

Τα σχόλια στην γλώσσα προγραμματισμού C μπορούν να αποτελούν μία γραμμή κωδικά εώς και πολλές. Για να 'ανοίξουμε' ένα σχόλιο αρκεί να χρησιμοποιήσουμε την Συμβολοσειρά /* και για να τα κλείσουμε χρησιμοποιούμε αντιστοίχως την */

Το κείμενο ανάμεσα σε αυτές τις συμβολοσειρές αγνοείται απο τον μεταγλωττιστή

  #include <stdio.h>

  /* Το σχόλιο αυτό είναι μία γραμμή */

  /* Το σχόλιο αυτό βρίσκεται
  σε πολλές
  γράμμες*/

  int main(void)
  {
    /* Η εντολή δεν θα εκτελεστεί */
