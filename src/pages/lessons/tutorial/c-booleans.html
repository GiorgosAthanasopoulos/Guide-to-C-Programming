Λογικές τιμές

Στις περισσότερες από τις γλώσσες σήμερα υπάρχουν τύποι δεδομένων ειδικά φτιαγμένοι για να εκπροσωπούνε λογικές τιμές, δηλαδή την τιμή αληθής ή ψευδής. Ο τύπος αυτός λέγεται boolean (έχει πάρει την ονομασία απο την μαθηματικό George Boole), ή απλά bool, και μπορεί να λάβει τις τιμές true ή false, οι οποίες παραδοσιακά αντιστοιχούν στις αριθμητικές τιμές 1 και 0. Στην C όμως τα πράγματα είναι λίγο διαφορετικά.

'Truthy' τιμές

Ας δούμε ένα παράδειγμα κώδικα 


  #include <stdio.h>

  int main()
  {
    if (5) 
    {
      printf("C");
    }

    return 0;
  }
Τι θα γίνει όταν τρέξουμε αυτό το πρόγραμμα; Θα τρέξει ή θα μας βγάλει κάποιο σφάλμα; Έχει νόημα να γράφουμε if (5); Η απάντηση είναι πως έχει νόημα και θα τρέξει κανονικά και θα εμφανιστεί το γράμμα 'C' στην οθόνη. Στην C δεν υπάρχει τύπος μεταβλητών bool, και συνεπώς δεν εξετάζει αν η έκφραση μέσα στις παρενθέσεις του if αποτιμούνται σε true ή false. Αντιθέτως, υπάρχουν τιμές που θεωρούνται 'truthy', δηλαδή δεν είναι το boolean true, όμως αν βρεθούν μέσα σε if θα θεωρηθούν αληθείς. Αυτές οι τιμές είναι κάθε σχήμα bit εκτός απο το 0. Συνεπώς, το 5 είναι διάφορο του 0 και συνεπώς θεωρείται αληθές. Τι θα κάνουμε όμως αν θέλουμε να χρησιμοποιήσουμε μία μεταβλητή ως bool; Στην βιβλιοθήκη stdbool.h υπάρχουν δύο macros, true και false, τα οποία αντιστοιχόυν προφανώς στις τιμές 1 και 0. Μπορούμε λοιπόν να δηλώσουμε μία μεταβλητή ως int και να της θέσουμε μία απο αυτές τις τιμές και πέρα απο την διαφορά στην μνήμη που καταναλώνει η διαφορά δεν είναι αισθητή με το bool.

Μιά εναλλακτική

Αν είστε συνηθισμένοι στην χρήση bool ή αν απλά θεωρείται οτι κάνει την σύνταξη προγραμμάτων πιο ξεκάθαρη, μία εναλλακτική είναι να χρησιμοποιήσετε το εξής: 


  enum Bool {
    False, True
  }

  Bool should_quit = False;
Το πρώτο γράμμα της κάθε λέξης είναι κεφαλαιό επειδή οι λέξεις bool, false και true είναι δεσμευμένες στην C++ και ίσως να θέλετε να μεταφέρετε κάποιο project σας σε αυτήν κάποια στιγμή.
