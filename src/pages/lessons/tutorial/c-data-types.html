Τύποι Δεδομένων

Όπως είδαμε στο μάθημα των μεταβλητών, οι μεταβλητές πρέπει να 'έχουν' κάποιον τύπο που τις χαρακτηρίζουν. Για παράδειγμα: 


  int i = -5;
  float f = 3.14f;
  double d = 6.28;
  char c = 'D';

Βασικοί τύποι δεδομένων

Τύπος Μέγεθος Περιγραφή
int 4 bytes Αποθηκεύει ακέραιους αριθμούς
float 4 bytes Αποθηκεύει δεκαδικούς αριθμούς με ακρίβεια 7 δεκαδικών ψηφιών
double 8 bytes Αποθηκεύει δεκαδικούς αριθμούς με ακρίβεια 16 δεκαδικών ψηφίων
char 1 bytes Αποθηκεύει χαρακτήρες που εμπίπτουν στο ASCII

Στον παραπάνω πίνακα μπορείτε να δείτε λεπτομέρειες για τον κάθε τύπο

Τροποποιήσεις τύπων

Μπορούμε να χρησιμοποιήσουμε κάποιες δεσμευμένες λέξεις που μας παρέχει η γλώσσα προκειμένου να αλλάξουμε ελαφρώς την λειτουργεία των τύπων που έχουμε εξετάσει ώς τώρα. Αρχικά, με εξαίρεση τους τύπους που χρησιμοποιούνται για την αποθήκευση δεκαδικών αριθμών, μπορούμε να χρησιμοποιήσουμε τις λέξεις signed και unsigned ώστε να πούμε στον μεταγλωττιστή πως η μεταβλητή ή μπορεί ή δεν μπορεί να λαμβάνει αρνητικές τιμές αντιστοίχως. Επίσης, μπορούμε να χρησιμοποιήσουμε τις λέξεις short και long ώστε να αλλάξουμε το μέγεθος κάποιον τύπων. Δείτε παρακάτω: 


  /* Πρέπει να είναι μη αρνητικό */
  unsigned int i = 5;

  /* Το signed μπορεί να παραληφθεί */
  signed int x = -4;

  unsigned char c = 'a';

  /* Είναι τουλάχιστον 4 bytes
  εξαρτάται απ'το μηχάνημα */
  long f = 3;

  /* Συνήθως 2 bytes */
  short s = 453; 

  /* ERROR */
  unsigned float f = 3;
