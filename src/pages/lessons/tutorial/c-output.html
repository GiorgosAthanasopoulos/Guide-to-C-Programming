Έξοδος

Σε κάθε πρόγραμμα που θα γράψετε θα πρέπει να εμφανίζεται κάποιο μήνυμα ή κάποια τιμή στην οθόνη, ειδάλλως δεν θα είναι ιδιαιτέρως χρήσιμο (εκτός και αν επιτελεί κάποια προχωρημένη λειτουργεία)

Εμφάνιση κειμένου

Όπως είδαμε και πρίν, μπορούμε να χρησιμοποιήσουμε την συνάρτηση printf που βρίσκεται μέσα στην βιβλιοθήκη stdio.h για να εμφανίσουμε κείμενο καθώς και τιμές μεταβλητών. Υπάρχουν και άλλες συναρτήσεις που εμφανίζουν συμβολοσειρές στην οθόνη, όμως θα εξετάσουμε μόνο αυτήν. 


  #include <stdio.h>

  int main()
  {
    printf("Hello, world!");
    return 0;
  }

Μπορούμε να βάλουμε μία δεύτερη κλήση της printf, και τότε θα παρατηρήσουμε ότι δεν εισάγεται νέα γραμμή μετά το τέλος του πρώτου μηνύματος 


  #include <stdio.h>
          
  int main()
  {
    printf("Hello, world!");
    printf(" same line");
    return 0;
  }

Νέα γραμμή

Σε όλες τις γλώσσες προγραμματισμού υπάρχουν ειδικοί χαρακτήρες οι οποίοι μεταλλάσσουν την έξοδο με ειδικό τρόπο. Οι χαρακτήρες γράφονται ως συνδυασμό του back-slash \ και ενός γράμματος. Αν βάλουμε n μετά το back-slash (δηλαδή \n) τότε όταν συναντηθεί αυτός ο χαρακτήρας κατα την εκτύπωση θα αλλάξει γραμμή η εκτύπωση. 


  #include <stdio.h>
                  
  int main()
  {
    printf("Hello, world!\n");
    printf(" different line");
    return 0;
  }

Δοκιμάστε και αυτό το πρόγραμμα να δείτε τι θα γίνει!
