Συμβολοσειρές (Strings)

Αν είχαμε την δυνατότητα να αποθηκεύαμε μονάχα έναν χαρακτήρα την φορά, θα ήταν πολύ δύσκολο να φτιάξουμε ένα χρήσιμο πρόγραμμα. Τι θα γινόταν αν θέλαμε για παράδειγμα να αναπτύξουμε ένα πρόγραμμα επεξεργασίας κειμένου; Για αυτόν τον λόγο έχουμε την δυνατότητα να αποθηκεύουμε ολόκληρες σειρές από σύμβολα (χαρακτήρες), οι οποίες μπορεί συνολικά να σχηματίζουν μία λέξη ή ένα ολόκληρο κείμενο

String Literals

Όπως υπάρχουν οι μεταβλητές με τιμή 6 και η σταθερή τιμή 6, υπάρχουν και μεταβλητές που αποθηκεύουν μία συμβολοσειρά (κατα κάποιο τρόπο) και υπάρχει και η συμβολοσειρά ως σταθερή τιμή. Αυτή την σταθερή τιμή την ονομάζουμε string literal και εντός των προγραμμάτων περιβάλλεται από διπλά εισαγωγικά 

 "This is a string literal";
Χωρίς να μπούμε σε πολλές λεπτομέρειες ακόμη για το πώς δουλεύουν αυτά που θα παρουσιάσουμε, ας δούμε πώς μπορούμε να αποθηκεύσουμε τους χαρακτήρες: 
  char* p = "Pointer to string literal";
  char s[] = "Array of chars";
Η διαφορά αυτή είναι πολύ σημαντική. Στην πρώτη περίπτωση, η οποιαδήποτε απόπειρα αλλαγής του περιεχομένου του δείκτη p οδηγεί σε απροσδιόριστη συμπεριφορά, που θα μπορούσε να οδηγήσει σε αντικανονική έξοδο του προγράμματος. Ουσιαστικά, δεν αποθηκεύεται όλη η συμβολοσειρά μέσα στην μεταβλητή, αλλά η μεταβλητή 'δείχνει' την διεύθυνση στην μνήμη του πρώτου χαρακτήρα. Αυτό είναι καλό επειδή οι χαρακτήρες των συμβολοσειρών βρίσκονται σε διαδοχικές θέσεις μνήμης, πράγμα που σημαίνει οτι αν γνωρίζουμε την διεύθυνση του πρώτου χαρακτήρα μίας συμβολοσειράς μπορούμε να βρούμε και τους επόμενους χαρακτήρες απλα προσπελάζοντας την μνήμη. Στην δεύτερη περίπτωση, παρόλο που θέτουμε ένα string literal στον πίνακα, μπορούμε να το μεταβάλλουμε εκ των υστέρων με τον ίδιο τρόπο που αλλάζουμε τις τιμές των θέσεων ενός πίνακα. Όμως, δεν μπορούμε να ξαναθέσουμε μία συμβολοσειρά στον πίνακα αφού τον έχουμε δημιουργήσει, όμως μπορούμε να το κάνουμε σε έναν δείκτη: 

  char s[] = "123"; /* ok */
  s = "321" /* error */;

  char* p = "123" /* ok */
  p = "321" /* ok */
Ένας τρόπος να θέσουμε μία τιμή εκ των υστέρων σε έναν πίνακα είναι με την χρήση της συνάρτησης strcpy που διαθέτει η βιβλιοθήκη string.h. Η συνάρτηση αυτή ΔΕΝ κάνει έλεγχο για το αν υπάρχει διαθέσιμος χώρος στον πίνακα για να θέσει την συμβολοσειρά, οπότε θεωρείται μη ασφαλής για χρήση, αλλα μπορούμε να το χρησιμοποιήσουμε για αρχή 

  #include <string.h>

  int main()
  {
    char s[] = "123"
    strcpy(s, "321");
    printf("%s", s); /* prints '321' */
    
    return 0;
  }
Υπάρχουν τρόποι να χρησιμοποιούμε συμβολοσειρές πιο δυναμικά, αλλά θα το παραλείψουμε καθώς και είναι πιο προχωρημένο
