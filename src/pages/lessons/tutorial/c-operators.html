Τελεστές

Οι τελεστές χρησιμοποιούνται μεταξύ μεταβλητών και σταθερών προκειμένου να ολοκληρώσουμε υπολογισμούς.

Αριθμιτικοί τελεστές

Υπάρχουν τελεστές οι οποίοι χρησιμοποιούνται για τον υπολογισμό κάποιας αριθμητικής τιμής. Τους τελεστές αυτούς τους παρουσιάζουμε παρακάτω:

Τελεστής Ονομασία Περιγραφή Παράδειγμα
+ Πρόσθεση Προσθέτει δύο τιμές x + y
- Αφαίρεση Αφαιρεί μία τιμή απο μιά άλλη x - y
* Πολλαπλασιασμός Πολλαπλασιάζει δύο τιμές x * y
/ Διαίρεση Διαιρεί μία τιμή με μία άλλη x / y
% Modulus Επιστρέφει το υπόλοιπο της διαίρεσης x % y
++ Αύξηση Αυξάνει την τιμή μιάς μεταβλητής κατά 1 ++x
-- Μείωση Μειώνει την τιμή μιάς μεταβλητής κατά 1 --x

Λογικοί και σχετικόι τελεστές

Άλλοι τελεστές χρησιμοποιούνται για να προσδιορίζουν τι σχέση υπάρχει μεταξύ δύο μεταβλητές

Τελεστής Περιγραφή Παράδειγμα
== Ισότητα x = y
< Μικρότερο από x < y
<= Μικρότερο ή ίσο x <= y
> Μεγαλύτερο από x > y
>= Μεγαλύτερο ή ίσο x >= y
|| Λογικό Ή x || y
&& Λογικό ΚΑΊ x && y
!= Διάφορο x != y
