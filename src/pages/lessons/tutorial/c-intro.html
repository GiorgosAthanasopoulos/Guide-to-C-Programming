Εισαγωγή

Λίγα λόγια για την γλώσσα

Η C δημιουργήθηκε το 1972 απο τον Dennis Ritchie στα Bell Labs και είναι μία απο τις πιο παλιές και ιστορικά σημαντικές γλώσσες του κόσμου και αποτελεί υπόβαθρο των σύγχρωνων γλωσσών προγραμματισμού. Χρησιμοποιεί κάποιους συντακτικούς κανόνες που λειτουργούν ως πρότυπο για τις περισσότερες γλώσσες που επήλθαν μετά απο αυτήν και επικεντρώνεται στην μέγιστη ταχύτητα εκτέλεσης ενός προγράμματος ενώ διατηρεί μία σχετική απλοότητα για τους ανθρώπους.

Γιατί να την χρησιμοποιήσω;

Όπως θα διαπιστώσετε κατά την εκμάθηση της γλώσσας, ή όπως ενδεχομένως να γνωρίζετε ήδη, η C σε σχέση με τις μοντέρνες γλώσσες θεωρείται σκληροπυρηνική και δεν διαθέτει τις ανέσεις που διαθέτουν πολλές γλώσσες σήμερα, όπως π.χ. η Python. Συνεπώς, είναι εύλογο να αναρωτηθεί κανείς γιατί να δυσκολευτεί να φτιάξει ένα πρόγραμμα σε C ενώ μπορεί να χρησιμοποιήσει ένα πολύ πιο εύχρηστο εργαλείο. Η αλήθεια είναι ότι σε πολλές περιπτώσεις συμφέρει πολύ περισσότερο να αναπτυχθεί ένα πρόγραμμα σε μία άλλη γλώσσα, γιατί έτσι θα αναπτυχθεί ταχύτερα και πιό εύκολα απο τον προγραμματιστή. Πέρα απο το ώφελος που κερδίζει κανείς απο την βαθεία κατανόηση της C, είναι εξαιρετικά χρήσιμη σε καταστάσεις που η ταχύτητα εκτέλεσης του προγράμματος είναι πάρα πολύ σημαντική, σε προγράμματα που επικοινωνούν με την συσκεύη στην οποία τρέχουν σε χαμηλό επίπεδο και για προγράμματα που τρέχουν σε ασυνήθιστες συσκευές.
