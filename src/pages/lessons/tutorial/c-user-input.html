Είσοδος (Input)

Όπως έχουμε δει, μπορούμε να χρησιμοποιήσουμε την συνάρτηση printf ώστε να εμφανίσουμε μηνύματα στην οθόνη του χρήστη. Τέτοια μηνήματα μπορεί να είναι π.χ. το αποτέλεσμα ενός πολύπλοκου υπολογισμού. Το πρόβλημα είναι όμως οτι αν οι παράμετροι ενός προγράμματος δεν μεταβάλλονται απο τον χρήστη, θα αποτελείται κάθε φορά ο ίδιος υπολογισμός και θα εμφανίζεται κάθε φορά το ίδιο αποτέλεσμα, γεγονός που καθιστά το εν λόγω πρόγραμμα όχι ιδιαιτέρως χρήσιμο. Μπορούμε όμως να χρησιμοποιήσουμε μία απο τις συναρτήσης εισόδου της C, για να λάβουμε κάποια τιμή που θα πληκτρολογήσει ο χρήστης κατα την εκτέλεση του προγράμματος. Μία απο αυτές τις συναρτήσεις είναι η scanf. Η scanf σταματάει το πρόγραμμα μέχρι ο χρήστης να πληκτρολογήσει κάτι και να πατήσει enter, και έπειτα συνεχίζει την εκτέλεση κανονικά. 

#include <stdio.h>

int main(void)
{
  int i;
  char c;
  scanf("%d %c", &i, &c);
  printf("%c %d", c, i);

  return 0;
}

Αν κατα την εκτέλεση του παραπάνω προγράμματος ο χρήστης πληκτρολογήσει 1 c τότε θα εκτυπώσουμε c 1 στην οθόνη. Μην ανησυχήσεις αν κάποια πράγματα φαίνονται Κινέζικα, θα τα αναλύσουμε τώρα. Αρχικά, όπως η printf σημαίνει print format, η scanf σημαίνει scan format, δηλαδή περνάμε ως πρώτη παράμετρο το format, δηλαδή το τι θα διαβάσει απτο πληκτρολόγιο το πρόγραμμα (δηλαδή αν θα διαβάσει αριθμό, χαρακτήρα κτλ), και έπειτα περνάμε ως παραμέτρους τις μεταβλητές που αντιστοιχόυν στο format. Πιο απλά, γράφοντας %d στην πρώτη θέση σημαίνει οτι η πρώτη παράμετρος που θα περάσουμε μετά το format θα είναι τύπου int, γράφοντας %c στην δεύτερη θέση σημαίνει οτι η δεύτερη παράμετρος που θα περάσουμε μετα το format θα είναι τύπου char κ.ο.κ.

Γιατί όμως γράφουμε &i αντί για i σκέτο; Η διαφορά αυτή είναι πολύ σημαντική, όμως δεν πρόκειται να το αναλύσουμε απο τώρα καθώς και είναι πιο προχωρημένο θέμα. Προς το παρόν, θα γράφετε &i και &c στις παραμέτρους, ακόμη και αν δεν το καταλαβαίνετε, γιατί ειδάλλως η συνάρτηση μπορεί να οδηγήσει στην ανώμαλη εξόδο του προγράμματος.

Άλλες συναρτήσεις εισόδου

Εντός του αρχείο stdio.h υπάρχουν δηλωμένες και άλλες συναρτήσεις για είσοδο που επιτελούν αμυδρώς διαφορετικές λειτουργείες. Αρχικά, για πολλούς λόγους η συνάρτηση scanf δεν θεωρείται ασφαλής και πρέπει να αποφεύγεται η χρήση της, όμως υπάρχει μία ανάλογη, η scanf_s (scan format safe), που μπορούμε να χρησιμοποιήσουμε. Άλλες συναρτήσεις είναι η fgets, fscanf, fgetc, κτλ.
