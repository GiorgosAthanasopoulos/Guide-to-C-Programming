Μαθηματικά

Όχι, δεν θα αναλύσουμε ολοκληρώματα και σειρές, δεν υπάρχει λόγος να πανικοβληθείτε, απλά θα παρουσιάσουμε μερικές συναρτήσεις που διαθέτει η βιβλιοθήκη των μαθηματικών της C για περίπλοκους μαθηματικούς υπολογισμούς (στην περίπτωσή μας ο όρος 'συνάρτηση' είναι και εύστοχος μαθηματικά)

Η Βιβλιοθήκη <math.h>

Πολλές φορές στην προγραμματιστική σας πορεία θα συναντήσετε την ανάγκη να κάνετε κάποιον λίγο πιό περίπλοκο υπολογισμό απο το 1 + 1. Δυστυχώς, η γλώσσα δεν διαθέτει ενθεματικούς τελεστές ώστε να υπολογίζουμε ημίτονα και τετραγωνικές ρίζες, κάτι που είναι απολύτως λογικό.

Δεν θα αναλύσουμε τι κάνει η κάθε συνάρτηση συγκεκριμένα επειδή δεν είμαστε εδώ για να σας μάθουμε μαθηματικά. Μπορείτε να δείτε όλες τις συναρτήσεις που περιέχει η βιβλιοθήκη εδώ. Θα δείξουμε όμως ένα παράδειγμα για να συνεχίσουμε την παράδοση. 

  #include <math.h>
  #include <stdio.h>

  int main()
  {
    printf("%lf", sqrt(3.14159265));

    return 0;
  }
Στο παραπάνω πρόγραμμα υπολογίιζουμε πολύ απλά την τετραγωνική ρίζα του π. Όσοι γνωρίζουν σχετικά καλά μαθηματικά δεν θα έχουν κανένα πρόβλημα ούτε να καταλάβουν τι κάνουν όλες αυτές οι συναρτήσεις ούτε θα δυσκολευτούν να τις χρησιμοποιήσουν. Δεν χρειάζεται να τις απομνημονεύσετε όλες, αρκεί να ξέρετε πού θα πρέπει να στραφείτε όταν θα έρθει η ώρα να κάνετε κάποιον μαθηματικό υπολογισμό, π.χ. όταν αναπτύσσετε ένα παιχνίδι και θα πρέπει να υπολογίσετε μία κυκλική απόσταση γύρω απο ένα σημείο
