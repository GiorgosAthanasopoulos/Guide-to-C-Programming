Μεταβλητές

Οι μεταβλητές είναι ένας τρόπος αν αναφερόμαστε σε μία θέση μνήμης με ένα όνομα, στην οποία μπορούμε να αποθηκεύουμε διαφορετικών τύπων δεδομένα. Στην γλώσσα προγραμματισμού C υπάρχουν μερικοί τύποι δεδομένων οι οποίοι της είναι εγγενείς. Όπως θα δούμε αργότερα, μπορούμε ως προγραμματιστές να ορίσουμε και δικούς μας τύπους.

Οι πιό βασικοί εγγενείς τύποι δεδομένων είναι οι εξής:

  • char - Ένας χαρακτήρας ο οποίος ανήκει στο ASCII
  • int - Ένας ακέραιος αριθμός
  • float - Ένας δεκαδικός αριθμός (αριθμός κινητής υποδιαστολής)
  • double - Ένας δεκαδικός αριθμός με διπλάσια ακρίβεια απ'τον float

Όπως θα δούμε παρακάτω υπάρχουν μερικές παραλλαγές τις οποίες μπορούμε να εφαρμόσουμε στους παραπάνω τύπους, καθώς και ένας επιπλέον τύπος (void) ο οποίος χρησιμοποιείται σε λίγο διαφορετικές περιπτώσεις.

Χρήση μεταβλητών

Για να χρησιμοποιήσουμε μία μεταβλητή στην C, πρέπει πρώτα από όλα να την δηλώσουμε. Τι σημαίνει αυτό; Σημαίνει οτι πρέπει να δηλώσουμε οτι υπάρχει μία μεταβλητή με ένα συγκεκριμένο όνομα ενός συγκεκριμένου τύπου. Πιο συγκεκριμένα: 

type name = value;

Όπου type είναι ένας από τους παραπάνω τύπους (π.χ. int) και value είναι μία τιμή η οποία αρμόζει στον εν λόγω τύπο (π.χ. η αριθμητική τιμή 4 όταν πρόκειται για μεταβλητή int). Να σημειωθεί οτι δεν είναι απαραίτητο να αναθέσουμε μία τιμή σε μία μεταβλητή με το που την δηλώσουμε, όμως γενικά συνιστάται.

Μπορούμε να αναθέσουμε τιμές σε μεταβλητές εγγενών τύπων ως εξής: 

int i = 5;
char c = 'a'; /* Είναι σημαντικό στους χαρακτήρες να χρησιμοποιούμε μονά εισαγωγικά (') και όχι διπλά (") */
float f = 5.4f;
double d = 1.2;

Μπορούμε επίσης να χρησιμοποιούμε μεταβλητές και ως μέρος μίας έκφρασης, (π.χ. αριθμητικής παράστασης) την οποία να αναθέσουμε σε μία άλλη μεταβλητή: 

int x = 5;
int y = 3;
int z = x + y; /* Το z θα έχει την τιμή 8 */
