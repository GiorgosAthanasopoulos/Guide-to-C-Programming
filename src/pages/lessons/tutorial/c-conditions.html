Συνθήκες

Η ροή ενός προγράμματος θα ήταν πολύ βαρετή αν ήταν πάντα σειριακή, δηλαδή αν η επόμενη εντολή που θα εκτελούταν ήταν πάντα η επόμενη. Πολλές φορές ανάλογα με την περίσταση μπορεί να θέλουμε να δράσουμε διαφορετικά και συνεπώς χρειαζόμαστε μία διακλάδωση στην εκτέλεση του προγράμματος.

H συνθήκη if

Η συνθήκη if (αν) υπάρχει σε πρακτικά κάθε γλώσσα που υπάρχει και είναι βασικό μέρος των προγραμμάτων. Χωρίς αυτήν δεν θα ήταν δυνατόν να φτιάχνουμε ακόμη και μερικά πολύ βασικά προγράμματα. 

  if (condition)
  {
    /* Do something here */
  }
Για παράδειγμα: 

  int x = 10;
  int y = 20;

  if (x + y >= 30)
  {
    printf("Greater than or equal");
  }

Η συνθήκη else

Η else θα λέγαμε ότι είναι κατα κάποιον τρόπο μία βοηθητική συνθήκη καθώς και δεν μπορεί να χρησιμοποιηθεί αυτόνομα. Χρησιμοποείται μετά απο μία συνθήκη if για να δείξει την εναλλακτική πορεία του προγράμματος αν δεν ισχύει η συνθήκη της if. 

  int x = 10;

  if (x < 10)
  {
    printf("Less than 10");
  }

  else
  {
    printf("Greater than 10");
  }
Η else μπορεί να έχει απευθείας την λέξη if μετά για να εξεταστεί μία άλλη συνθήκη. Αυτό θα ήταν ισοδύναμο με το να βάζαμε ένα άλλο if μέσα στις αγκύλες μετά το else.
