Break και Continue

Break

Είδαμε προηγουμένως την χρήση του break ως έναν τρόπο ώστε να βγούμε απο το switch και να μην εκτελεστούν οι υπόλοιπες περιπτώσεις. Το break μπορεί επίσης να χρησιμοποιηθεί για να σταματήσουμε την εκτέλεση μιάς επανάληψης. 


  for (int i = 0; i < 5; ++i)
  {
    if (i == 4)
    {
      break;
    }       
    
    printf("%d\n", i);
  }
Μετά την εκτέλεση αυτής της επανάληψης θα εμφανιστούν οι αριθμοί 1 2 3, επειδή όταν το i φτάνει την τιμή 4, χρησιμοποιούμε την εντολή break και η επανάληψη λήγει οριστικά και συνεχίζει κανονικά η εκτέλεση του προγράμματος.

Continue

Η εντολή continue μας επιτρέπει να προσπεράσουμε την εκτέλεση της τρέχουσας επανάληψης και να εξετάσουμε την συνθήκη ξανά. Ας το δούμε πρακτικά για να γίνει πιο ξεκάθαρο 


    for (int i = 0; i < 5; ++i)
    {
      if (i == 4)
      {
        continue;
      }

      printf("%d\n", i);
    }
Δοκιμάστε να εκτελέσετε αυτή την επανάληψη και θα δείτε ότι εμφανίζονται όλοι οι αριθμοί εκτός από τον αριθμό 4, επειδή τότε κάνουμε continue και όλες οι εντολές μετά απο αυτήν δεν εκτελούνται για μία μόνο φορά.
