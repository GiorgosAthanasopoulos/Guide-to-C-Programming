Η επανάληψη For

Όταν γνωρίζουμε εκ των προτέρων πόσες φορές θέλουμε να εκτελεστεί ένα κομμάτι κώδικα τότε είναι σκόπιμο να χρησιμοποιήσουμε την επανάληψη for αντί για την while. Η σύνταξη της for έχει ως εξής: 

  for (statement 1; statement 2; statement 3)
  {
    /* do something */
  }
Η εντολή 1 εκτελείται μία φορά πριν την αρχή της επανάληψης. Συνήθως χρησιμοποιείται για την αρχικοποίηση κάποιας τιμής, η οποία συνήθως είναι ο μετρητής της επανάληψης. Η εντολή 2 δεν είναι εντολή παρά μία συνθήκη η οποία ελέγχεται πριν συνεχιστεί η επανάληψη. Συνήθως εκεί ελέγχουμε αν ο μετρητής της επανάληψης κυμένεται στο εύρος τιμός που επιθυμούμε. Η εντολή 3 συνήθως χρησιμοποιείται για να αυξηθεί/μειωθεί ο μετρητής της επανάληψης και εκτελείται κάθε φορά που τελειώνει η επανάληψη. Να σημειωθεί πως ΔΕΝ είναι απαραίτητο να χρησιμοποιήσετε την κάθε εντολή για τον σκοπό που αναφέραμε, αρκεί να κατανοείτε πώς χρησιμοποιείται η κάθε μία από το πρόγραμμα.

Ένα παράδειγμα επανάληψης με την επανάληψη for είναι το εξής: 

  for (int i = 0; i < 5; ++i)
  {
    printf("%d\n", i);
  }
