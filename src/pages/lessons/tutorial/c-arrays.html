Πίνακες

Οι πίνακες χρησιμοποιούνται για να αποθηκεύουν πολλές τιμές μαζί σε μία μεταβλητή, αντί να χρησιμοποιούμε πολλές μεταβλητές για κάθε τιμή που θέλουμε να αποθηκεύσουμε

Για να δηλώσουμε έναν πίνακα, πρέπει να διευκρινήσουμε τον τύπο, να διευκρινήσουμε το όνομα και μετά να βάζουμε τετράγωνες αγκύλες και μέσα σε αυτές το μέγεθος του πίνακα ως ακέραιο. π.χ. 

int nums[5];
Έτσι δημιουργήσαμε έναν πίνακα με 5 θέσεις. Προσοχή! Ο χώρος για τους πίνακες δεσμεύεται στην αρχή του προγράμματος, πράγμα που σημαίνει οτι πρέπει να ξέρουμε το μέγεθός του ενώ συνθέτουμε τον κώδικα. Δηλαδή δεν μπορούμε να κάνουμε κάτι τέτοιο: 
  int c;
  scanf("%d", &c);
  int arr[c]; /* oxi! */

Μπορούμε να αρχικοποιήσουμε έναν πίνακα βάζοντας τιμές ανάμεσα σε αγκύλες οι οποίες χωρίζονται με κόμματα π.χ. 

 int arr[4] = { 1, 2, 3, 4 }
Και για να αλλάξουμε την τιμή του ν-οστού στοιχείου αλλάζουμε την θέση ν-1 του πίνακα. Δηλαδή για να αλλάξουμε την πρώτη θέση κάνουμε 
 arr[0] = 3;
Επειδή εδώ η θέση 0 είναι το πρώτο στοιχείο του πίνακα
