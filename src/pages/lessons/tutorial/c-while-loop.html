While loop

Οι επαναλήψεις μπορούν να εκτελέσουν ένα κομμάτι κώδικα εφόσον η συνθήκη της ικανοποιείται. Οι επαναλήψεις είναι χρήσιμες επειδή μας γλιτώνουν χρόνο, κόπο, μειώνουν τα σφάλματα και κάνουν τον κώδικα πιο εύκολο στην ανάγνωση. Η πρώτη επανάληψη που θα δούμε είναι η while.

Η while επαναλαμβάνει την εκτέλεση του κώδικα μέσα στις αγκύλες της όσο (while) η συνθήκη είναι αληθής. Η σύνταξη έχει ως εξής: 

  while (condition)
  {
    /* Do something */
  }

Ας δούμε ένα παράδειγμα στο οποίο εκτυπώνονται οι τιμές απο το 1 εώς το 5 


  int i = 1;

  while (i <= 5)
  {
    printf("%d\n", i);
    ++i;
  }
