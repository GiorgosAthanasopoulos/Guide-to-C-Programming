Structs

Πολλές φορές κατα τον προγραμματισμό είναι πάρα πολύ χρήσιμο να συνδυάζουμε πολλές μεταβλητές μέσα σε μία έτσι ώστε να επιτυγχάνουμε τους σκοπούς μας πιο εύκολα και καθαρά.

Ένα απλοϊκό παράδειγμα

Φανταστείτε ότι συντάσσετε ένα πρόγραμμα το οποίο ζωγραφίζει σχήματα στην οθόνη. Μέσα σε αυτό το πρόγραμμα θα δημιουργήσετε συναρτήσεις οι οποίες λαμβάνουν ως παραμέτρους πληροφορίες σχετικά με αυτά τα σχήματα, όπως την τοποθεσία τους στην οθόνη και το μέγεθός τους. Έστω τώρα πως θέλουμε να ζωγραφίσουμε την κοινή περιοχή δύο ορθογωνίων. Αυτό σημαίνει πως θα πρέπει να περάσουμε σε μία συνάρτηση τις πληροφορίες 2 ορθογονίων, δηλαδή την τεταγμένη του, την τετμημένη του, το πλάτος του και το ύψος του. Συνεπώς δύο τέτοια ορθογώνια θα απαιτήσουν 8 διαφορετικά ορίσματα για να λειτουργήσει η συνάρτηση. Αυτό απο μόνο του δημιουργεί ένα χάος, αλλα για τώρα χρησιμοποιούμε αυτή την τεχνική και έχουμε μία συνάρτηση με 8 παραμέτρους. Έστω τώρα πως αργότερα αποφασίζουμε πως θελουμε το κάθε ορθογώνιο επίσης να έχει χρώμα, που σημαίνει ότι θα πρέπει να βάλουμε τουλάχιστον άλλες 3 μεταβλητές των ενός byte για κάθε ορθογώνιο (rgb), που σημαίνει ότι η συνάρτησή μας τώρα όχι μόνον παίρνει 14 ορίσματα (γεγονός που είναι απολύτως χαοτικό), αλλά θα πρέπει να αλλάξουμε τον ορισμό της συνάρτησης καθώς και κάθε κλήση της συνάρτησης μέσα στο πρόγραμμα. Είναι ξεκάθαρο πως αυτό δεν μπορεί να συνεχιστεί. Για αυτόν τον λόγο μπορούμε να χρησιμοποιήσουμε structs, και να κάνουμε την ζωή μας εύκολη. 

            typedef unsigned char byte_t;

            struct Color {
              byte_t r, g, b;
            }

            struct Size {
              unsigned int cx, cy;
            }

            struct Point {
              int x, y;
            }

            struct Rectangle {
              struct Point pos;
              struct Size size;
              struct Color color;
            }

            void DrawIntersection(struct Rectangle a, struct Rectangle b)
            {
              /* ... */
            }
Τώρα, όχι μόνον εκφράζεται με απόλυτη σαφήνεια ο σκοπός του κάθε μέλους του ορθογωνίου, αλλά η οποιαδήποτε αλλαγή στην δομή μπορεί να εφαρμοστεί χωρίς να αλλάξει κανένα άλλο στοιχείο. Με άλλα λόγια, προσφέρει και ένα στοιχείο αφαιρετικότητας
