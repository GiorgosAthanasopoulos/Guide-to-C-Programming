Μάθηματα στην γλώσσα προγραμματισμού C

Καλωσήρθατε! Κάνατε ένα καλό πρώτο βήμα στην πορεία σας στο να μάθετε καλύτερο προγραμματισμό με το να έλθετε σε αυτήν την σελίδα. Στα μαθήματα που ακολουθούν θα εξετάσουμε την γλώσσα προγραμματισμού C σε ένα επαρκές βάθος έτσι ώστε μετά την ολοκλήρωση όλων των μαθημάτων και των ασκήσεων που σας παρέχουμε να μπορείτε να αρχίσετε να προγραμματίζετε και συνεπώς να μαθένετε σε πραγματικό βάθος την C.

Η κάθε σελίδα περιέχει τις πληροφορίες που κρίναμε ως ομάδα οτι είναι απολύτως απαραίτητες για μία καλή κατανόηση της κάθε ιδέας που παρουσιάζεται. Σκοπός δεν είναι να μπορείτε να φτιάξετε αμέσως ό,τι περίπλοκη ιδέα σκεφτείτε, αλλά να κατανοήσετε τα εφόδια που σας παρέχει η γλώσσα και τον τρόπο που λειτουργεί έτσι ώστε σιγά σιγά να αποκτήσετε την ιδιότητα να υλοποιείται πράγματα καθαρά, ορθά και αποτελεσματικά.

Έπειτα απο διάφορες παραγράφους θα παραθέτουμε και αποσπάσματα κώδικα, τα οποία δεν θα είναι απαραίτητα αυτόνομα προγράμματα, απλά εξυπηρετούν τον σκοπό της επίδειξης ορισμένων χαρακτηριστικών και ιδεών της γλώσσας.
