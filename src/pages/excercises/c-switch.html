'Ασκήσεις σε Switch

Άσκηση 1

Εισαγάγετε τα μέρη που λείπουν για να συμπληρώσετε την ακόλουθη δήλωση switch. 

          int day = 2;
          switch(){
             1:
              printf("%s", "Saturday");
              break;
             2:
              printf("%s", "Sunday");
              ;
          }

Άσκηση 2

Συμπληρώστε τη δήλωση switch και προσθέστε τη σωστή λέξη-κλειδί στο τέλος για να καθορίσετε κάποιον κώδικα που θα εκτελεστεί εάν δεν υπάρχει αντιστοίχιση πεζών-κεφαλαίων στη δήλωση switch. 

          int day = 4;
          switch(){
             1:
              printf("%s", "Saturday");
              break;
             2:
              printf("%s", "Sunday");
              ;
            :
              printf("%s", "Weekend");
          }
