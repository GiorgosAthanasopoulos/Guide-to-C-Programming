Άσκηση 1

Εισαγάγετε το τμήμα του κώδικα που λείπει παρακάτω για να εμφανιστεί το "Hello World!". 


  int main(){
    ("Hello World!");
    return 0; 
  }

Εισαγάγετε μια νέα γραμμή μετά το "Hello World", χρησιμοποιώντας έναν ειδικό χαρακτήρα: 


  int main() {
    printf("Hello World!");
    printf("I am learning C");
    return 0;
  }
