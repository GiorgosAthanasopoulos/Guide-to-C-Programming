Ασκήσεις σε επαναλήψεις

Οι επαναλήψεις μπορεί να δυσκολεύουν πολλούς που προγραμματίζουν για πρώτη φορά, για αυτό είναι σημαντικό να εξασκθηούμε σε αυτές ώστε να μπορούμε να φανταστούμε την ροή του προγράμματος κατά την εκτέλεση επαναλήψεων έτσι ώστε να μπορούμε να τις συνθέτουμε σωστά.

Άσκηση 1

Συμπληρώστε τα παρακάτω κενά έτσι ώστε να εκτυπώνονται οι ακέραιοι αριθμοί από το 1 μέχρι το 5 


  int i = 1;
   (i < 6) {
    printf("%d\n", i);
    i  1;
  }

Άσκηση 2

Συμπληρώστε τα παρακάτω κενά έτσι ώστε να εκτυπώνονται όλοι οι ακέραιοι αριθμοί απο το 1 εώς το 6 


  int i = 1;
   {
    printf("%d\n", i);
    i += ;
  }  (i < 6);

Άσκηση 3

Συμπληρώστε τα παρακάτω κενά έτσι ώστε να εμφανιστεί η λέξη 'Yes' 5 φορές στην οθόνη. 

  
   (int i = 0; i < 5; i) {
    puts("");
  }

Άσκηση 4

Συμπληρώστε το κενό έτσι ώστε η επανάληψη να διακόπτεται όταν το i αποκτά την τιμή 5 

          
  for (int i = 0; i < 10; ++i) {
    if (i == 5) {
      ;
    }
    printf("%d\n", i);
  }

Άσκηση 5

Συμπληρώστε το κενό έτσι ώστε να μην εκτυπωθεί η τιμή 4


  for (int i = 0; i < 10; ++i) {
    if (i == 4) {
      ;
    }
    printf("%d\n", i);
  }
