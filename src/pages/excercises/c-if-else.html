Ασκήσεις σε If Else

Η χρήση της δομής ελέγχου if-else είναι μία απο τις πιο συνηθισμένες και σημαντικές σε όλες τις γλώσσες προγραμματισμού, για αυτο και θα πρέπει να εξοικειωθούμε με αυτή

Άσκηση 1

Συμπληρώστε τα κενά έτσι ώστε να εμφανιστεί το μήνυμα "Hello, world" στην οθόνη αν το x είναι μικρότερο από το y 


  int x = 50;
  int y = 10;
   (x  y) {
    puts("Hello, world");
  }

Άσκηση 2

  int y = 50;
   (x  y) {

Άσκηση 3

Συμπληρώστε τα κενά έτσι ώστε το παρακάτω πρόγραμμα να εμφανίζει "Yes" αν το x ισούται με το y, ειδάλλως "No"
