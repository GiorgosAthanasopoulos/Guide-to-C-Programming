Switch

Η εντολή switch χρησιμοποιείται ώστε να επιλεχθεί ένα κομμάτι κώδικα το οποίο θα εκτελεστεί βάσει την αποτίμιση μιάς έκφρασης. Η σύγκριση γίνεται αυστηρά με σταθερές τιμές. Για παράδειγμα: 

  int x = 10;

  switch (x)
  {
  case 5:
    printf("5");
    break;

  case 10:
    printf("10");
    break;

  default:
    printf("Neither");
    break;
  }

Ας αναλύσουμε το παραπάνω κομμάτι κώδικα. Αρχικά, γράφουμε switch και βάζουμε την τιμή την οποία εξετάζουμε μέσα στην παρένθεση. Μετά, χρησιμοποιούμε την λέξη case για την κάθε περίπτωση και την τιμή που αντιστοιχεί στην περίπτωση ακριβώς μετά. Δείτε πως μετά απο κάθε κλήση της printf χρησιμοποιούμε την δεσμευμένη λέξη break. Αν δεν την χρησιμοποιούσαμε, τότε μετά την εκτέλεση της μίας περίπτωσης, η εκτέλεση θα συνέχιζε μέχρι να συναντηθεί κάποιο άλλο break. Δηλαδή: 


  int x = 5;

  switch (x)
  {
  case 5:
    printf("5");

  case 10:
    printf("10");
  }
Αν εκτελέσετε αυτόν τον κώδικα, θα εμφανιστεί και ή τιμή 5 και η τιμή 10 στην οθόνη. Αυτές οι περιπτώσεις ονομάζονται fallthrough.
